计算公式 (IEC 60751):
对于 t ≥ 0°C: R(t) = R₀(1 + At + Bt²)
对于 t < 0°C: R(t) = R₀[1 + At + Bt² + C(t-100)t³]
其中: R₀ = 100 Ω, A = 3.9083×10⁻³, B = -5.775×10⁻⁷, C = -4.183×10⁻¹²
